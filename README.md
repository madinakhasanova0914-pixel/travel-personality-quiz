# Travel Personality Quiz 🌍

An interactive personality quiz that recommends a travel destination based on your preferences. Answer 3 questions and discover whether you are a Maldives or Rome type of traveler!

## About
Built as part of the Girls Who Code JavaScript certificate program. I chose a travel theme because I love exploring different cultures and destinations.

## What It Does
- Presents 3 travel preference questions with image choices
- Tracks user selections using JavaScript
- Calculates and displays a personalized destination result
- Includes a restart button to retake the quiz

## Concepts Used
- HTML structure and semantic markup
- CSS flexbox layout and styling
- JavaScript event listeners
- DOM manipulation
- Score tracking with variables
- Conditional logic for results
